##java
```
java [ options ] class [ argument...  ]

java [ options ] -jar file.jar [ argument...  ]

参数：
       options        命令行参数
       class          需要加载的class
       file.jar       用`-jar file.jar` 加载需要的jar包
       argument       传递给main方法的参数
```
